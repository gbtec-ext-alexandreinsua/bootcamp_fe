Objectivo
    El objectivo de este ejercicio es crear un sitio web con dos páginas.

Requisitos  
    Interfaz de usuario
        Hay que crear dos páginas. 
        La primera página muestra un mensaje de bienvenida y tiene un botón para mostrar un menú.
        La segunda página muestra una lista de promociones.

    Funcionalidad
        Ambas páginas tiene un menú en una cabecera.
        El item Inicio lleva a la página de bienvenida.
        El item Promociones lleva a la pádina de promociones.

    Transiciones y animaciones
        Los enlaces tienen un estilo para el estado hover
        El botón "Ver menú" tiene estilos para el estado hover y active

    Fuente y tema
        La fuente es Arial
        El color de fondo base es #dbceae y variaciones de tono 
        El resto son colores habituales #fff, #eee, #333
    
    El footer tiene un padding de 1.5rem

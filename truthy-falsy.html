Valores Truthy y Falsy en JavaScript

En JavaScript, los valores no siempre se limitan a true y false. Dependiendo del contexto, algunos valores se comportan como si fueran verdaderos o falsos en expresiones booleanas, como las usadas en condicionales if. Estos valores se conocen como truthy y falsy.

¿Qué son los Valores Truthy?

Los valores truthy son aquellos que, cuando se evalúan en una expresión booleana, se comportan como true. A continuación se presentan algunos ejemplos: 

// Ejemplos de valores truthy
if ("JavaScript") console.log("Las cadenas no vacías son truthy");
if (" ") console.log("Las cadenas con espacios en blanco");
if (42) console.log("Cualquier número distinto de 0 es truthy");
if ([]) console.log("Incluso un array vacío es truthy");
if ({}) console.log("Un objeto vacío también es truthy");
if (function() {}) console.log("Las funciones son truthy");
if (Infinity) console.log("Infinity y -Infinity son truthy");

¿Qué son los Valores Falsy?

Por otro lado, los valores falsy son aquellos que se evalúan como false en un contexto booleano. Hay solo unos pocos valores que se consideran falsy en JavaScript: 

// Ejemplos de valores falsy
if (!false) console.log("false es falsy");
if (!0) console.log("0 es falsy");
if (!"") console.log("Una cadena vacía es falsy");
if (!null) console.log("null es falsy");
if (!undefined) console.log("undefined es falsy");
if (!NaN) console.log("NaN es falsy");

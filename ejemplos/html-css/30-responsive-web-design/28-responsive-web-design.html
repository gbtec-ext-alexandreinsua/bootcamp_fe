Responsive Web Design

¿Qué es el Responsive Web Design?

El Responsive Web Design (RWD) o Diseño Web Responsivo es un enfoque de diseño y desarrollo web que busca garantizar que las páginas web se vean y funcionen bien en una amplia variedad de dispositivos y tamaños de pantalla. Desde teléfonos móviles, tabletas, hasta monitores de escritorio, el diseño responsivo se adapta automáticamente al dispositivo del usuario, proporcionando una experiencia óptima sin necesidad de versiones separadas del sitio.

Principios del Responsive Web Design

    Principales técnicas del Responsive Web Design

    1. Optimización del viewport: Configurar el meta viewport en el HTML para dispositivos móviles. 
      <meta name="viewport" content="width=device-width, initial-scale=1.0">
    2. Diseño basado en layouts flexibles: Utilizar CSS Grid o Flexbox para crear estructuras adaptativas. Se utiliza una u otra opción dependiendo si se necesita un layout de una o dos dimensiones. 
      .container {
                    display: flex;
                    flex-wrap: wrap;
                    }
                    .item {
                    flex: 1 1 auto;
                    }
    3. Uso de Media Queries: Define estilos específicos para diferentes tamaños de pantalla. Se usan de forma complementaria con una disposición de elementos flexible. En general se utiliza para definir un tamaño máximo de un elemento en dispositivos de grandes dimensiones. 
      
                    @media (width > 1200px) {
                    .container {
                    max-width: 1200px;
                    }
                    }
    4. Imágenes y videos responsivos: Las imágenes deben ajustarse automáticamente al tamaño de su contenedor. Tambien es convieniente revisar el tipo de visualización el elemento, porque puede ser interesante establecerla en inline-block o block. 
      img {
                    max-width: 100%;
                    height: auto;
                    }
    5. Tipografía adaptable: Usar unidades relativas como em, rem o vw. Es especialmente inportante en los títulos. 
      body {
                    h2: 2vw;
                    }
    6. Menús de navegación adaptativos: Crear menús que cambien su apariencia en dispositivos móviles. De manera general, en dispositivos móviles hay elementos como menús o sidebars que están ocultos por defecto y se muestran a partir un punto de corte tipo tablet. 
      @media (width > 768px) {
                    .menu {
                    display: block;
                    }
                    .hamburger {
                    display: none;
                    }
                    }

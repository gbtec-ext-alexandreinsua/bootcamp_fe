Pseudo-clases en CSS

¿Qué son las Pseudo-clases?

En CSS, las pseudo-clases son palabras clave que se utilizan para definir un estado especial de un elemento. Permiten aplicar estilos en función de características dinámicas, como si un enlace ha sido visitado, si un campo de formulario está enfocado o si un elemento es el primero dentro de su contenedor.

Estructura de una Pseudo-clase


selector:pseudo-clase {
    propiedad: valor;
}

Pseudo-clases comunes

  • Interacción del usuario:
    • :hover: Aplica estilo cuando el cursor está sobre un elemento.
    • :focus: Se aplica cuando un elemento está enfocado.
    • :active: Aplica estilo mientras un elemento está siendo presionado.
  • Estados de formulario:
    • :disabled: Aplica estilo a elementos de formulario deshabilitados.
    • :checked: Aplica estilo a elementos seleccionados, como casillas de verificación o radio buttons.
    • :valid / :invalid: Estilo según si los datos de entrada cumplen las reglas de validación.
  • Estructurales:
    • :nth-child(n): Selecciona el elemento enésimo hijo.
    • :first-child / :last-child: Aplica estilo al primer o último hijo de un elemento contenedor.

Atributos de Campos de Texto en Formularios

Un campo de formulario de tipo texto (<input type="text">) en HTML puede llevar una variedad de atributos que definen su comportamiento, apariencia y validación. Aquí te presento una lista de los atributos más comunes:

Atributos Básicos

  • type: Define el tipo del campo de entrada. En este caso, su valor es "text". 
    <input type="text">
  • name: Especifica el nombre del campo para identificarlo en los datos enviados al servidor. 
    <input type="text" name="usuario">
  • value: Define el valor inicial del campo. 
    <input type="text" value="Hola Mundo">
  • id: Proporciona un identificador único al campo. 
    <input type="text" id="nombre">
  • class: Asigna clases CSS para estilos personalizados. 
    <input type="text" class="form-control">
  • placeholder: Muestra un texto de ayuda cuando el campo está vacío. 
    <input type="text" placeholder="Introduce tu nombre">
  • readonly: Hace que el campo sea de solo lectura. 
    <input type="text" value="Solo lectura" readonly>
  • disabled: Desactiva el campo, impidiendo que el usuario lo edite o interactúe con él. 
    <input type="text" disabled>

Atributos de Validación

  • required: Hace que el campo sea obligatorio. 
    <input type="text" required>
  • maxlength: Define el número máximo de caracteres que se pueden ingresar. 
    <input type="text" maxlength="20">
  • minlength: Define el número mínimo de caracteres que se deben ingresar. 
    <input type="text" minlength="5">
  • pattern: Especifica una expresión regular que el valor debe cumplir. 
    <input type="text" pattern="[A-Za-z]+" title="Solo letras permitidas">
  • size: Establece un ancho visual del campo (en caracteres). 
    <input type="text" size="30">

Atributos de Autocompletado y Sugerencias

  • autocomplete: Controla si el navegador puede autocompletar el campo. 
    <input type="text" autocomplete="on">

    Valores posibles:

    • "on": Habilita el autocompletado (predeterminado).
    • "off": Deshabilita el autocompletado.
  • list: Asocia el campo a una lista de opciones predefinidas usando la etiqueta <datalist>. 
    
<input type="text" list="opciones">
<datalist id="opciones">
    <option value="Opción 1">
    <option value="Opción 2">
</datalist>

Atributos de Ayuda y Descripción

  • title: Proporciona información adicional como un tooltip. 
    <input type="text" title="Introduce tu nombre completo">

Atributos de Control de Texto

  • spellcheck: Habilita o deshabilita la corrección ortográfica. 
    <input type="text" spellcheck="true">
  • dir: Establece la dirección del texto (izquierda a derecha o derecha a izquierda). 
    <input type="text" dir="rtl">

Ejemplo Completo


<form>
    <label for="nombre">Nombre:</label>
    <input 
        type="text" 
        id="nombre" 
        name="nombre" 
        placeholder="Introduce tu nombre" 
        maxlength="30" 
        minlength="3" 
        required 
        pattern="[A-Za-z ]+"
        title="Solo se permiten letras y espacios">
</form>

Principales Funciones en CSS

En CSS, las funciones son herramientas que permiten realizar cálculos, transformar valores o manipular propiedades de manera dinámica y flexible. A continuación, se mencionan las principales funciones que se utilizan en CSS:

calc()

Permite realizar cálculos matemáticos dentro de las propiedades CSS. Puedes usarla para sumar, restar, multiplicar o dividir unidades de medida, lo que proporciona gran flexibilidad al trabajar con tamaños y márgenes. 

 div { 
  width: calc(100% - 20px); 
 }

Aquí, el ancho del div será el 100% del contenedor menos 20 píxeles.

var()

La función var() permite utilizar variables CSS, facilitando la reutilización de valores y la creación de temas dinámicos. 

:root { 
  --primary-color: #3498db; 
 } 

button {
            background-color: var(--primary-color); 
 }

En este ejemplo, se define una variable --primary-color en el selector :root y luego se usa en los botones. 

div 
 { 
  background-color: hsl(120, 100%, 50%);   /* Verde */
            
  color: hsla(0, 100%, 50%, 0.3);   /* Rojo con 30% de opacidad */ 
 }

clamp()

La función clamp() se utiliza para establecer un valor que se ajusta dentro de un rango entre un valor mínimo y máximo. 

div 
 { 
  font-size: clamp(14px, 5vw, 24px); 
 }

En este caso, el tamaño de la fuente se ajusta dinámicamente con un valor mínimo de 14px, un valor máximo de 24px, y un valor intermedio basado en el ancho de la ventana (5vw).

min() y max()

Estas funciones permiten seleccionar el valor mínimo o máximo entre varios valores. 

div 
 { 
  width: min(50%, 300px);   /* El ancho será el menor de 50% o
            300px */ 
 } 

div 
 { 
  width: max(200px, 10vw);   /* El
            ancho será el mayor de 200px o 10% del ancho de la ventana */ 
 }

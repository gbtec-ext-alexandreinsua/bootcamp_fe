Propiedad Transform en CSS

La propiedad transform permite mover, rotar, escalar o inclinar un elemento, creando efectos visuales increíbles. Aquí te mostramos cómo funciona con ejemplos prácticos.

1. Translate (Mover el elemento)

La función translate mueve un elemento desde su posición original:

Mover a (100px, 20px)

2. Rotate (Rotar el elemento)

La función rotate gira un elemento alrededor de su centro:

Rotar 45°

3. Scale (Cambiar tamaño)

La función scale cambia el tamaño de un elemento:

Escalar 2x

4. Skew (Inclinar el elemento)

La función skew inclina un elemento en los ejes X o Y:

Inclinar 30° en X

5. Matrix (Combinación avanzada)

La función matrix combina varias transformaciones en una sola:

Matrix simple

¡Experimenta con estas transformaciones y crea efectos geniales en tus proyectos!

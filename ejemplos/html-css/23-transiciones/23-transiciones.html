Lección: Introducción a las Transiciones en CSS

¿Qué son las transiciones en CSS?

Las transiciones en CSS permiten cambiar gradualmente las propiedades de estilo de un elemento a lo largo del tiempo, en lugar de hacerlo instantáneamente. Esto mejora la experiencia del usuario al hacer que las interacciones sean más suaves y atractivas.

Ejemplo básico:

button {
    background-color: blue;
    transition: background-color 0.5s ease;
}

button:hover {
    background-color: red;
}

Explicación: Al pasar el cursor sobre el botón (hover), el color de fondo cambia de azul a rojo en 0.5 segundos.

Propiedades clave de las transiciones

Las transiciones se definen con la propiedad transition. Sus componentes principales son:

  • transition-property: Especifica la propiedad que cambiará (por ejemplo, background-color, transform, opacity). Usa all para incluir todas las propiedades aplicables.
  • transition-duration: Define cuánto tiempo tarda el cambio en segundos o milisegundo (por ejemplo, 500ms, 2s).
  • transition-timing-function: Controla cómo se acelera o desacelera la transición. Los valores más frecuentes son:
    • ease: Cambio suave.
    • linear: Velocidad constante.
    • ease-in: Comienza lento y acelera.
    • ease-out: Comienza rápido y desacelera.
    • ease-in-out: Comienza lento, acelera y desacelera.
  • transition-delay: Retrasa el inicio de la transición (por ejemplo, 1s).

Sintaxis completa:

selector {
    property: value;
    transition: property duration timing-function delay;
}

selector:pseudoclass{
    property: newValue;
}

Ejemplo práctico:

div {
    width: 100px;
    height: 100px;
    background-color: blue;
    transition: width 1s ease-in-out, background-color 0.5s linear;
}

div:hover {
    width: 200px;
    background-color: red;
}

Más ejemplos prácticos

Ejemplo 1: Cambio de Opacidad

img {
    opacity: 1;
    transition: opacity 0.5s ease-in-out;
}

img:hover {
    opacity: 0.5;
}

Movimiento con transform

.box {
    width: 100px;
    height: 100px;
    background-color: green;
    border: 3px solid darkgreen;
    border-radius: 5px;
    transition: transform 3s ease-out;
}

.box:hover {
    transform: translateX(100px) rotate(365deg);
}

Buenas Prácticas

  • Menos es más: Usa transiciones sutiles para evitar distracciones.
  • Evita transiciones en propiedades críticas: Como width y height, ya que pueden ser menos eficientes. Usa transform en su lugar.
  • Compatibilidad: Asegúrate de probar en diferentes navegadores.

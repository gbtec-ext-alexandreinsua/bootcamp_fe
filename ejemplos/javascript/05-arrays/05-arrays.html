Arrays en JavaScript

Esta referencia cubre los conceptos básicos de los arrays en JavaScript, incluyendo la creación y manipulación de datos en arrays. En futuras lecciones, ampliaremos estos temas con bucles adicionales y métodos avanzados para trabajar con arrays.

Concepto de Array

Un array es una estructura de datos que permite almacenar múltiples valores en una sola variable. Cada valor se encuentra en una posición específica, conocida como índice.

Por ejemplo:

const frutas = ["manzana", "banana", "pera"];

También se puede usar el constructor de arrays:

const numeros = new Array(1, 2, 3, 4, 5);

En JavaScript los arrays pueden ser unidimensionales o multidimensionales. Es muy común usar arrays bidimensionales para representar un plano con ejes x e y: 

const tabla = [
        [1, 2, 3], // Fila 1
        [4, 5, 6], // Fila 2
        [7, 8, 9] // Fila 3
        ];

También es común usar arrays tridimensionales para representar un plano de tres dimensiones.

Nota: En JavaScript, un array puede contener datos de diferentes tipos en la misma estructura. Por ejemplo, podríamos tener un array que contenga números, cadenas de texto y booleanos: 

const mezcla = [1, "hola", true, null];

Acceso a Elementos en un Array

Los elementos en un array se acceden usando su índice (empezando desde 0). Ejemplo: 

const colores = ["rojo", "verde", "azul"];
        console.log(colores[0]); // Imprime: "rojo"
        console.log(colores[2]); // Imprime: "azul"

Si intentas acceder a un índice que no existe en el array, obtendrás undefined.

El acceso a los elementos de un array multidimensional se realiza mediante el uso de combinación de índices: 


        console.log(tabla[0][1]); // Accede al elemento en la primera fila, segunda columna: imprime "2"
        console.log(tabla[2][0]); // Accede al elemento en la tercera fila, primera columna: imprime "7"

Métodos estáticos

El método Array.isArray() determina si el valor pasado es un Array.


        Array.isArray([1, 2, 3]); // true
        Array.isArray({ foo: 123 }); // false
        Array.isArray("foobar"); // false
        Array.isArray(undefined); // false

Operaciones Comunes con Arrays

Agregar elementos

push() agrega un elemento al final del array:

colores.push("amarillo"); // ["rojo", "verde", "azul", "amarillo"]

unshift() agrega un elemento al inicio del array:

colores.unshift("morado"); // ["morado", "rojo", "verde", "azul", "amarillo"]

Eliminar elementos

pop() elimina el último elemento:

colores.pop(); // ["morado", "rojo", "verde", "azul"]

shift() elimina el primer elemento:

colores.shift(); // ["rojo", "verde", "azul"]

Modificar elementos

Cambiar un valor en una posición específica:

colores[1] = "cyan"; // ["rojo", "cyan", "azul"]

Obtener la longitud del array

La propiedad .length devuelve el número de elementos en el array. Por ejemplo: 

console.log(colores.length); // 3

Recorrer un Array

Existen varias formas de recorrer un array en JavaScript. Vamos a usar el siguiente array como ejemplo:

const numeros = [1, 2, 3, 4];

Usando un bucle for

El bucle for es una de las formas más básicas y comunes.

const numeros = [1, 2, 3, 4];

        for (let i = 0; i <numeros.length; i++) {
        console.log(`Índice ${i}: ${numeros[i]}`);
        }

Cómo funciona:

  • let i = 0: Comenzamos desde el primer índice del array.
  • i <numeros.length: El bucle sigue mientras i sea menor que la longitud del array.
  • i++: Incrementamos i después de cada iteración para avanzar al siguiente elemento.
  • Accedemos a los elementos usando numeros[i].

Usando for...of

El bucle for...of se introdujo en ES6 y es más sencillo para recorrer arrays directamente por sus valores.

const numeros = [1, 2, 3, 4];

        for (const numero of numeros) {
        console.log(numero);
        }

Usando forEach

El método forEach permite recorrer cada elemento y ejecutar una función sobre él. Este método recibe una función de callback que gestiona el bucle.

Esta función recibe dos parámetros. El primero el es elemento de la interacción actual y es obligatorio. El segundo es el índice de la interacción actual y es optativo.

const numeros = [1, 2, 3, 4];

        numeros.forEach((numero, indice) => {
        console.log(`Índice ${indice}: ${numero}`);
        });

Usando un bucle while

El bucle while es menos común, pero también sirve para recorrer arrays.

const numeros = [1, 2, 3, 4];
        let i = 0;

        while (i < numeros.length) {
        console.log(`Índice ${i}: ${numeros[i]}`);
        i++;
        }

Diferencia entre for..of y forEach

Las diferencias entre ambos bucles son las siguientes:

  • for..of es un bucle nativo del lenguaje, mientras que forEach es un método específico de arrays.
  • forEach no puede romperse con break o continue, mientras que for..of sí.
  • forEach ofrece acceso al índice y al array completo, mientras que for..of itera solo sobre los valores.

Métodos avanzados de arrays en JavaScript

Los métodos avanzados de arrays en JavaScript permiten realizar operaciones más sofisticadas sobre los datos almacenados. Estos métodos son fundamentales para trabajar con datos de manera funcional y eficiente.

Método de ordenación sort()

Descripción: El método sort() ordena los elementos de un array in place (modifica el array original) y los devuelve ordenados según una función de comparación.

array.sort([compareFunction])

Parámetros:

  • compareFunction (opcional): Una función que define el orden de los elementos.
    • Si se omite, los elementos se convierten a cadenas y se ordenan según el valor Unicode.
    • Para ordenar numéricamente:
      • (a, b) => a - b // Orden ascendente
            (a, b) => b - a // Orden descendente

La función de comparación tiene que tener la siguiente estructura:

function compare(a, b) {
        if (a es menor que b según criterio de ordenamiento) {
        return -1;
        }
        if (a es mayor que b según criterio de ordenamiento) {
        return 1;
        }
        // a debe ser igual b
        return 0;
        }

Ejemplo:

const numeros = [5, 1, 10, 3];
        numeros.sort((a, b) => a - b);
        console.log(numeros); // [1, 3, 5, 10]

Método de filtrado filter()

Descripción: El método filter() crea un nuevo array con los elementos que cumplan la condición definida en una función de callback.

array.filter(callback(elemento, index))

Parámetros:

  • callback (obligatorio): Función que se ejecuta para cada elemento del array. Debe devolver true para mantener el elemento o false para descartarlo.
    • element (obligatorio): El elemento actual del array.
    • index (opcional): El índice del elemento actual.

Ejemplo:

const numeros = [1, 2, 3, 4, 5];
        const mayoresQueTres = numeros.filter(num => num > 3);
        console.log(mayoresQueTres); // [4, 5]

Metodo de transformación map()

Descripción: El método map() crea un nuevo array con los resultados de aplicar una función a cada elemento del array original.

array.map(callback(elemento, index))

Parámetros:

  • callback (obligatorio): Función que se aplica a cada elemento del array.
    • element (obligatorio): El elemento actual del array.
    • index (opcional): El índice del elemento actual.

Ejemplo:

const numeros = [1, 2, 3, 4];
        const cuadrados = numeros.map(num => num * num);
        console.log(cuadrados); // [1, 4, 9, 16]

Método de reducción reduce()

Descripción: El método reduce() aplica una función acumuladora sobre los elementos del array para reducirlo a un único valor.

array.reduce(callback(accumulator, elemento, index), initialValue)

Parámetros:

  • callback (obligatorio): Función que se ejecuta sobre cada elemento del array.
    • accumulator (obligatorio): El valor acumulado tras cada iteración.
    • element (obligatorio): El elemento actual del array.
    • index (opcional): El índice del elemento actual.
  • initialValue (opcional): El valor inicial del acumulador. Si se omite, el primer elemento del array se usa como valor inicial y la iteración empieza desde el segundo elemento.

Ejemplo:

const numeros = [1, 2, 3, 4];
        const suma = numeros.reduce((acc, num) => acc + num, 0);
        console.log(suma); // 10

Método de búsqueda find()

Descripción: El método find() devuelve el primer elemento que cumpla la condición definida en una función de callback. Si no encuentra ninguno, devuelve undefined.

array.find(callback(elemento, index))

Parámetros:

  • callback (obligatorio): Función que se ejecuta para cada elemento.
    • element (obligatorio): El elemento actual.
    • index (opcional): El índice del elemento actual.

Ejemplo:

const numeros = [1, 2, 3, 4];
        const encontrado = numeros.find(num => num > 2);
        console.log(encontrado); // 3

Método de verificación some()

Descripción: El método some() verifica si al menos un elemento cumple con la condición definida en una función de callback. Devuelve true si encuentra uno o más elementos que cumplan la condición; de lo contrario, devuelve false.

array.some(callback(elemento, index))

Parámetros:

  • callback (obligatorio): Función que se ejecuta para cada elemento.
    • element (obligatorio): El elemento actual.
    • index (opcional): El índice del elemento actual.

Ejemplo:

const numeros = [1, 2, 3, 4];
        const hayMayoresQueTres = numeros.some(num => num > 3);
        console.log(hayMayoresQueTres); // true

Método de verificacón every()

Descripción: El método some() verifica si todos los elementos del array cumplen la la condición definida en una función de callback. Devuelve true si todos los elementos cumplen la condición; de lo contrario, devuelve false.

array.some(callback(elemento, index ))

Parámetros:

  • callback (obligatorio): Función que se ejecuta para cada elemento.
    • element (obligatorio): El elemento actual.
    • index (opcional): El índice del elemento actual.

Ejemplo:

const numeros = [5, 8, 9, 5];
      const hayMayoresQueTres = numeros.every(num => num > 3);
      console.log(hayMayoresQueTres); // true

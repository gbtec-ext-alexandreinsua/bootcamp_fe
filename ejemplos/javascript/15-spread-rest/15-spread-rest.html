Operadores Spread y Rest en JavaScript

Operador Spread

El operador spread (...) expande elementos de un iterable (como un array, un objeto, o incluso una cadena de texto) en lugares donde se esperan elementos individuales.

Esto significa que toma los elementos iterables y los procesa uno a uno, como elementos individuales.

Usos principales del operador spread

  1. Copiar arrays u objetos: 
    const arr = [1, 2, 3];
const copy = [...arr]; // [1, 2, 3]
  2. Combinar arrays u objetos: 
    const arr1 = [1, 2];
const arr2 = [3, 4];
const combined = [...arr1, ...arr2]; // [1, 2, 3, 4]
  3. Pasar elementos como argumentos: 
    const numbers = [4, 5, 6];
Math.max(...numbers); // 6
  4. Desestructurar con propiedades adicionales: 
    const obj = { a: 1, b: 2 };
const newObj = { ...obj, c: 3 }; // { a: 1, b: 2, c: 3 }

Ejemplo completo con arrays:

const arr1 = [1, 2, 3];
const arr2 = [4, 5];
const combined = [...arr1, ...arr2];
console.log(combined); // [1, 2, 3, 4, 5]

Operador Rest

El operador rest (...) agrupa múltiples elementos en una sola colección. Se utiliza en la definición de funciones, al desestructurar arrays u objetos, y en otros contextos donde se necesitan manejar múltiples valores como una unidad.

Es decir, es el operador contrario al operador Spread. Toma varios elementos individuales y los agrupa en cuna colleccion

Usos principales del operador rest

  1. En funciones para aceptar argumentos variables: 
    function sum(...numbers) {
    return numbers.reduce((acc, num) => acc + num, 0);
}
console.log(sum(1, 2, 3, 4)); // 10
  2. En desestructuración de arrays: 
    const [first, ...rest] = [10, 20, 30, 40];
console.log(first); // 10
console.log(rest);  // [20, 30, 40]
  3. En desestructuración de objetos: 
    const { a, ...rest } = { a: 1, b: 2, c: 3 };
console.log(a);    // 1
console.log(rest); // { b: 2, c: 3 }

Diferencias clave entre spread y rest

Aspecto Spread Rest
Propósito Expandir elementos Agrupar elementos
Contexto típico Al llamar funciones, crear arrays, clonar u combinar objetos Al definir funciones o al desestructurar
Ejemplo común [...array] function(...args)

Ejemplo combinado

function processValues(first, ...rest) {
    console.log("Primero:", first);
    console.log("El resto:", rest);
}

const arr = [10, 20, 30, 40];
processValues(...arr);

// Salida:
// Primero: 10
// El resto: [20, 30, 40]

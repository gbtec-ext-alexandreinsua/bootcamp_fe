Clase Set en JavaScript

¿Qué es un Set en JavaScript?

Un Set es una colección de valores que:

  • No permite duplicados: Cada valor en un Set debe ser único. Si intentas agregar un valor que ya existe, no se añadirá.
  • Es iterable: Puedes recorrer los elementos del Set en el orden en el que fueron añadidos.

El propósito principal de un Set es garantizar que no existan elementos duplicados dentro de la colección.

Características principales de Set

  • No admite duplicados: Cualquier valor añadido al Set que ya esté presente será ignorado.
  • Orden de inserción: Los elementos de un Set mantienen el orden en el que se añadieron.
  • Comparación basada en igualdad estricta (===): Los valores se comparan usando la igualdad estricta, excepto para objetos, que se consideran únicos aunque tengan el mismo contenido.
  • Claves y valores son iguales: En un Set, los valores son tanto las claves como los valores.

Sintaxis para crear un Set

const miSet = new Set();

También puedes inicializarlo con un iterable, como un array:

const miSet = new Set([1, 2, 3, 4, 4]); // Los valores duplicados serán eliminados
console.log(miSet); // Set(4) { 1, 2, 3, 4 }

Métodos principales de Set

1. add(valor)

Agrega un valor al Set. Si el valor ya existe, no se añade. 

const miSet = new Set();
miSet.add(1);
miSet.add(2);
miSet.add(1); // Ignorado
console.log(miSet); // Set(2) { 1, 2 }

2. delete(valor)

Elimina un valor del Set. Devuelve true si el valor existía y se eliminó, o false si no existía. 

miSet.delete(2); // true
console.log(miSet); // Set(1) { 1 }

3. has(valor)

Devuelve true si el valor existe en el Set, o false si no existe. 

console.log(miSet.has(1)); // true
console.log(miSet.has(3)); // false

4. clear()

Elimina todos los elementos del Set.

miSet.clear();
console.log(miSet); // Set(0) {}

5. size

Devuelve el número de elementos en el Set.

const otroSet = new Set(["a", "b", "c"]);
console.log(otroSet.size); // 3

Iteración sobre un Set

1. Usando for...of:

const miSet = new Set(['a', 'b', 'c']);
for (const valor of miSet) {
  console.log(valor); // 'a', 'b', 'c'
}

2. Usando forEach:

miSet.forEach(valor => {
  console.log(valor);
});

3. Métodos de iteración:

  • keys(): Devuelve un iterador con los valores (en un Set, las claves y valores son iguales).
  • values(): También devuelve un iterador con los valores.
  • entries(): Devuelve un iterador con pares [valor, valor] (por compatibilidad con Map). 
const miSet = new Set([10, 20, 30]);

for (const valor of miSet.keys()) {
  console.log(valor); // 10, 20, 30
}

for (const entrada of miSet.entries()) {
  console.log(entrada); // [10, 10], [20, 20], [30, 30]
}

Ejemplo práctico: Eliminar duplicados de un array

Una de las aplicaciones más comunes de Set es eliminar duplicados de un array: 

const numeros = [1, 2, 3, 4, 4, 5, 5, 6];
const unicos = [...new Set(numeros)];
console.log(unicos); // [1, 2, 3, 4, 5, 6]

Diferencias entre Set y Array

Característica Set Array
Duplicados No permite duplicados. Permite duplicados.
Métodos de búsqueda Usa has(valor). Usa includes(valor).
Orden de inserción Mantiene el orden de inserción. También mantiene el orden.
Uso típico Colección única de valores. Colección de valores diversos.

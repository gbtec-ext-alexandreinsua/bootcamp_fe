Acceso a Elementos del DOM en JavaScript

Esta lección cubre el acceso a elementos del DOM en JavaScript mediante el uso del objeto document, métodos de selección como getElement y querySelector, y el uso de la notación de punto. Aquí aprenderemos cómo interactuar con los elementos de una página web de manera dinámica.

¿Qué es el DOM?

El DOM (Modelo de Objetos del Documento, por sus siglas en inglés Document Object Model) es una representación estructurada de una página web en forma de árbol de nodos. Cada elemento HTML en el documento, junto con los atributos y el contenido, es representado como un nodo en este árbol.

¿Para qué sirve el DOM?

El DOM permite que JavaScript y otros lenguajes puedan acceder y manipular el contenido y la estructura de una página web. Gracias al DOM, podemos cambiar dinámicamente el contenido, estilo y estructura de una página después de que esta ha sido cargada en el navegador.

El objeto document

En JavaScript, el objeto document es nuestra puerta de entrada al DOM. Podemos usar este objeto para acceder a elementos específicos de la página y manipularlos según necesitemos.

Ejemplo:

console.log(document); // Muestra el contenido completo del DOM en la consola

Ejemplo de Estructura del DOM

Considera este HTML simple:


        <html>
          <head>
            <title>Página Ejemplo</title>
          </head>
          <body>
            <h1>¡Hola, Mundo!</h1>
            <p>Este es un párrafo de
        ejemplo.</p>
          </body>
        </html>

Este código HTML se representa en el DOM de la siguiente manera:

  • html
    • head
      • title: "Página Ejemplo"
    • body
      • h1: "¡Hola, Mundo!"
      • p: "Este es un párrafo de ejemplo."

El Objeto document

El objeto document es la entrada principal para acceder al DOM en JavaScript. Representa toda la página cargada en el navegador y proporciona métodos y propiedades que permiten interactuar con los elementos HTML.

Ejemplo básico:

console.log(document); // Muestra el contenido completo del DOM

Podemos usar el objeto document para acceder a diferentes partes del DOM mediante notación de punto, métodos getElement o métodos querySelector.

Acceso a Elementos del DOM

Para seleccionar un elemento en el dom el documento tiene que estar cargo cualquier script de js que acceda a dom debe estar invocado DESPUES DEL BODY.

A. Acceso mediante la Notación de Punto

Algunos elementos comunes del DOM, como <title>, <body>, o <head>, pueden accederse directamente mediante la notación de punto.

Ejemplos:

console.log(document.title); // Obtiene el título de la página console.log(document.body); // Obtiene el contenido del elemento <body> console.log(document.documentElement); // Obtiene el nodo <html> completo

B. Acceso mediante los Métodos getElement

Los métodos getElement permiten seleccionar elementos específicos en el DOM, basándose en identificadores únicos (id), clases (class) o etiquetas (tag).

1. getElementById

Este método selecciona un único elemento usando su atributo id, que debe ser único en toda la página.

Ejemplo:

const $titulo = document.getElementById("titulo-principal"); console.log(titulo); // Muestra el elemento con id "titulo-principal" titulo.textContent = "Nuevo Título Principal";

2. getElementsByClassName

Este método selecciona todos los elementos que tienen una clase específica y devuelve una colección HTML (similar a un array).

Ejemplo:

const $items = document.getElementsByClassName("item"); console.log(items); // Muestra todos los elementos con la clase "item" items[0].style.color = "red";

3. getElementsByTagName

Este método selecciona todos los elementos que comparten el mismo nombre de etiqueta y devuelve una colección HTML.

Ejemplo:

const $parrafos = document.getElementsByTagName("p"); console.log(parrafos); // Muestra todos los elementos <p> parrafos[1].textContent = "Párrafo actualizado";

C. Acceso mediante querySelector y querySelectorAll

Los métodos querySelector y querySelectorAll permiten seleccionar elementos usando selectores CSS, como id, class, o cualquier selector CSS más complejo.

1. querySelector

Este método selecciona el primer elemento que coincide con el selector especificado.

Ejemplo:

const $primerTexto = document.querySelector(".texto"); console.log(primerTexto); // Muestra el primer <p> con la clase "texto" primerTexto.style.fontWeight = "bold";

2. querySelectorAll

Este método selecciona todos los elementos que coinciden con el selector especificado y devuelve una NodeList (similar a un array).

Ejemplo:

const $textos = document.querySelectorAll("div p"); console.log(textos); // Muestra todos los <p> dentro de <div> textos.forEach(texto => texto.style.color = "blue");

Acceso y Modificación a Atributos de Elementos HTML

En JavaScript, es posible acceder y manipular los atributos de un elemento HTML de diferentes formas, dependiendo de si se trata de atributos comunes, personalizados, estilos inline o clases CSS. A continuación, se explican los métodos principales para realizar estas operaciones.

1. Usando la "Nomenclatura del Punto"

La "nomenclatura del punto" es una forma directa de acceder a algunos atributos de un elemento HTML, como id, className o title. Solo funciona con atributos específicos que tienen propiedades asociadas en el objeto DOM.

Ejemplo:


<div id="miDiv" class="clase1" title="Ejemplo de título"></div>

const $elemento = document.getElementById("miDiv");
console.log($elemento.id);         // Imprime: "miDiv"
console.log($elemento.className);   // Imprime: "clase1"
console.log($elemento.title);       // Imprime: "Ejemplo de título"

2. Accediendo a Todos los Atributos de un Elemento

Para obtener todos los atributos de un elemento, puedes usar la propiedad attributes, que devuelve una colección de todos los atributos del elemento. Es útil para iterar sobre todos los atributos sin importar su nombre.

Ejemplo:


cont $elemento = document.getElementById("miDiv");

for (let attr of elemento.attributes) {
console.log(attr.name + " = " + attr.value);
}

3. Usando getAttribute() y setAttribute()

Para acceder o modificar cualquier atributo, puedes usar los métodos getAttribute() y setAttribute(). Esto es especialmente útil para atributos que no están disponibles mediante la nomenclatura del punto.

Ejemplo:


const $elemento = document.getElementById("miDiv");

console.log($elemento.getAttribute("id"));       // "miDiv"
console.log($elemento.getAttribute("class"));    // "clase1"
console.log($elemento.getAttribute("title"));    // "Ejemplo de título"

// Cambiar el valor del atributo `title`
elemento.setAttribute("title", "Nuevo título");
console.log($elemento.title);                    // "Nuevo título"

4.Accediendo a Atributos de Datos (Data Attributes)

HTML permite agregar atributos personalizados usando el prefijo data-. Estos atributos se pueden acceder y modificar con la propiedad dataset.

Ejemplo:


<div id="miDiv" data-rol="usuario" data-estado="activo"></div>

cont $elemento = document.getElementById("miDiv");

console.log($elemento.dataset.rol);       // "usuario"
console.log($elemento.dataset.estado);    // "activo"

// Cambiar el valor del atributo `data-estado`
elemento.dataset.estado = "inactivo";
console.log($elemento.dataset.estado);    // "inactivo"

5. Acceder y Modificar Estilos Inline (Propiedad style)

La propiedad style permite acceder y modificar los estilos inline de un elemento. Cada propiedad CSS puede manipularse directamente en JavaScript.

Ejemplo:


<div id="miDiv" style="color: red; font-size: 16px;"></div>

cont $elemento = document.getElementById("miDiv");

// Acceder a propiedades de estilo
console.log(elemento.style.color);          // "red"
console.log(elemento.style.fontSize);       // "16px"

// Modificar estilos inline
elemento.style.color = "blue";
elemento.style.fontSize = "20px";

6. Acceder y Modificar Clases CSS

Las clases CSS de un elemento se pueden manipular mediante la propiedad className o el objeto classList. Cada opción permite agregar, eliminar o alternar clases de manera efectiva.

Usando className

La propiedad className permite acceder y asignar todas las clases como un solo string.

Ejemplo:


cons $elemento = document.getElementById("miDiv");

// Obtener las clases como string
console.log(elemento.className);    // "clase1 clase2"

// Cambiar todas las clases
elemento.className = "clase3 clase4";
console.log(elemento.className);    // "clase3 clase4"

Usando classList

El objeto classList proporciona métodos como add(), remove(), toggle() y contains() para manipular clases individualmente.

Ejemplo:


const $elemento = document.getElementById("miDiv");

// Agregar una clase
elemento.classList.add("clase3");    // Ahora tiene "clase3"

// Remover una clase
elemento.classList.remove("clase1"); // Quita "clase1"

// Alternar una clase (agrega si no está, elimina si está)
elemento.classList.toggle("clase2"); // Quita "clase2" si está, o la agrega si no está

// Verificar si el elemento tiene una clase específica
console.log(elemento.classList.contains("clase3")); // true

Bucles en JavaScript

Introducción

En programación, un bucle es una estructura que permite repetir un bloque de código varias veces, dependiendo de una condición. En JavaScript, los bucles son fundamentales para realizar tareas repetitivas, como recorrer listas o ejecutar operaciones iterativas.

Tipos de Bucles en JavaScript

JavaScript ofrece varias formas de crear bucles, cada una diseñada para resolver situaciones específicas:

  • Bucle for
  • Bucle while
  • Bucle do...while
  • Bucle for...in
  • Bucle for...of

1. Bucle for

El bucle for se utiliza cuando conoces de antemano cuántas veces quieres iterar. Su sintaxis incluye tres partes principales: inicialización, condición, y actualización. 


for (inicialización; condición; actualización) {
    // Código a ejecutar
}

Ejemplo:


for (let i = 0; i < 5; i++) {
    console.log("Iteración:", i);
}

2. Bucle while

El bucle while se utiliza cuando no se conoce el número exacto de iteraciones, pero sí una condición que debe cumplirse para continuar. 


while (condición) {
    // Código a ejecutar
}

Ejemplo:


let contador = 0;
while (contador < 3) {
    console.log("Contador:", contador);
    contador++;
}

3. Bucle do...while

El bucle do...while garantiza que el código dentro del bloque se ejecutará al menos una vez, independientemente de la condición. 


do {
    // Código a ejecutar
} while (condición);

Ejemplo:


let numero = 0;
do {
    console.log("Número:", numero);
    numero++;
} while (numero < 3);

4. Bucle for...in

El bucle for...in recorre las propiedades enumerables de un objeto. 


for (let propiedad in objeto) {
    // Código a ejecutar
}

Ejemplo:


const persona = { nombre: "Juan", edad: 30 };
for (let propiedad in persona) {
    console.log(\`\${propiedad}: \${persona[propiedad]}\`);
}

5. Bucle for...of

El bucle for...of recorre los valores de un objeto iterable, como arrays, cadenas o mapas. 


for (let valor of iterable) {
    // Código a ejecutar
}

Ejemplo:


const frutas = ["Manzana", "Pera", "Plátano"];
for (let fruta of frutas) {
    console.log("Fruta:", fruta);
}

Comparación de Bucles

Bucle Cuándo Usarlo
for Cuando conoces el número exacto de iteraciones.
while Cuando no sabes cuántas veces iterar, pero tienes una condición clara.
do...while Cuando necesitas al menos una ejecución del bloque antes de evaluar.
for...in Cuando necesitas recorrer las propiedades de un objeto.
for...of Cuando necesitas recorrer los valores de un iterable (arrays, strings).
